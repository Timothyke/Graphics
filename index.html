Print-Ready File Guidelines

Already have a design? Make sure it’s print-ready:

  • Use CMYK color mode (not RGB)
  • Resolution: 300 DPI
  • Include 3mm bleed on all sides
  • Outline fonts or embed them
  • Save as PDF/X-1a or high-quality PDF

Need help? Our designers can fix or create your files!
